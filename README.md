### 管理后台系统

#### 技术栈
* React @16.13.1
* React-Redux @7.2.0
* React-Router @5.1.2
* Redux @3.7.2
* webpack @4.42.0
* antd @4.1.4
* 百度地图sdk
* Echarts

#### 项目启动

* yarn install 安装依赖
* yarn start 启动开发环境
* yarn build 项目打包

#### 代码提交规范
* husky 提交代码 git commit -m 触发钩子
* commitlint 配置 commintlint.config.js 设置配置
* docs,chore,feat,fix,merge,perf,refactor,revert,style,test
